# 5GC API Requester
* This project is developed by python 3.x.
* Testing Environment: Free5GC
* Usage
	```bash
	$ python requester.py
	```
	1. Input the valid IP address
	2. Select the testing network function

## Functionality
* `requester.py`
	* The main program
	* Send request to 5GC API
* `src`
	* The url of 5GC API
* `param.py`
	* Default parameter for 5GC API
